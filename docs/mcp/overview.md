# MCP Client Configuration

AutoMobile MCP is designed to be run in STDIO mode in production settings like workstations and CI automation.

```shell
npx -y auto-mobile@latest
```



```json
{
  "mcpServers": {
    "AutoMobile": {
      "command": "npx",
      "args": ["-y", "auto-mobile@latest"]
    }
  }
}
```
