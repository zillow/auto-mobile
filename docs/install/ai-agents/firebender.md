# Firebender MCP Config

```json
{
  "mcpServers": {
    "AutoMobile": {
      "command": "npx",
      "args": ["-y", "@kaeawc/auto-mobile@latest"]
    }
  }
}
```

If your Android SDK is not on `PATH`, add `ANDROID_HOME` (or `ANDROID_SDK_ROOT`/`ANDROID_SDK_HOME`) to the `env` map.
