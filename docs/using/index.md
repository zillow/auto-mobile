## Getting Started

AutoMobile can be used in several ways:

1. **Interactive Exploration** - Use AI agents to explore your app and discover issues
2. **Automated Testing** - Write reproducible UI tests that run in CI/CD
3. **Performance Monitoring** - Track frame rates, startup times, and transitions
4. **Accessibility Auditing** - Validate contrast ratios and touch target sizes

Each guide in this section focuses on a specific use case with practical examples.
