# Screen Transition

Screen transitions should be smooth and responsive. Janky transitions hurt user experience and make your app feel slow. Use AI to measure and optimize how quickly your app transitions between screens.

Make sure to read [the overview guide for performance analysis](index.md).

**Example Prompts**

> Navigate to <feature-x\> in the <app-name\> <Android\iOS\> app and measure the screen transition performance entering it from different destinations. Highlight and investigate any transitions that are performance outliers.
>
> Navigate to <feature-x\> in the <app-name\> <Android\iOS\> app, quickly switch back and forth between <screen-x\> and <screen-y\> and measure the screen transition performance in both directions.
